## Usage

### Clone

```sh
git clone git@github.com:keitoliveira/ultimate-pontte.git
cd ultimate-pontte
npm install
```

### Install dependencies

```sh
npm install
```

### Run in developer mode

```sh
npm start
```

### Run for production

```sh
npm run build
```
